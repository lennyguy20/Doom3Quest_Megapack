This is a megapack of modes for Doom3Quest v1.2.1
-------------------------------------------------

Xpak399.pk4 is a modifed version of pak399.pk4 (it begins with X to overwrite all custom weapons and screens from the mods).
In the pack only guis/mainmenu.gui is modified (mods screen). This might be redone to support newer version of Doom3Quest.

About Doom3Quest: https://www.doom3quest.com/
